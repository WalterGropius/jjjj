# LINE EVENTS

Landing page for LINE EVENTS s.r.o. — events, prostor, zážitek.

Built with Next.js 14 (App Router), TypeScript, Tailwind CSS and Framer Motion.

## Development

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Build

```bash
npm run build && npm start
```

## Brand

- LINE RED `#ff3427`
- LINE BLUE `#240f5a`
- LINE WHITE `#ffffff`
- Display: Helvetica Now Display Bold (web fallback: Inter Tight, Helvetica Neue)
